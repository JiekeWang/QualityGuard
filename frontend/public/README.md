## Login Background Image

To use a custom login background image:

1. Place your image file in this directory (e.g., `login-bg.jpg` or `login-bg.png`)
2. Update the image path in `Login.tsx` or configure it via Settings page

Recommended image format: JPG or PNG
Recommended size: 1920x1080 or higher for best quality
